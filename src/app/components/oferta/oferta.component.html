About us
“PRMSoftware” is a company created to deliver expert services and solutions for big and enterprise sized companies. It’s primary rule and goal is to create complexed and effective solutions for complicated problems in modern IT world. Keeping it in mind there are two more rules which creates company manifest fundamentals: “There is always a solution for a problem no matter how it is complicated. Solution does not have to be complicated same way problem is. Try to not shoot a fly with a tank first.” “Your work should be a joy not a duty. If it’s not a joy you will fail your duties. For 25 years we gathered a unique and broad set of knowledge necessary for designing and creating solutions. We do not believe in specialization and targeting for certain OS. Current world demands flexibility, we are aware we have to design multiplatform solutions. So for us there are no borders , write your code and run it everywhere it’s needed. 
        Our technology stack is :
        For programming :
        -Go/Python/Java
        -Javascript/Typescript
        -React/Angular
        For design :
        -Business analysis using modern research technologies
        user experience of process analysis
        -UI designs
        For clouds :
        -Openstack - Triple/Redhat with all features you want
        -Azure – Design as you wish, deploy as design.
        OS types :
        -All
